  1. regla de css y sintaxis

    • selector {
    propiedad: valor;
}

  2. aplicando css a html

    • estilos en linea 
          <p style="color: green;">hola soy un parrafo verde</p>
    • estilos internos 
          <head>
        <style>
            p {
                color: green;
            }
        </style>
    </head>
    <body>
        <p>soy un parrafo verde</p>
    </body>
    • estilos externos 
          <head>
        <link rel="stylesheet" href="style.css">
    </head>
    <body>
        <p>soy un parrafo verde</p>
    </body>

  3. selectores

    • elemento html 
          p {
        font-size: 30px;
    }
    • class 
          .parrafo {
        color: lightcoral;
    }
    • id 
          #parrafo-id {
        background-color: gray;
    }

    • hijo decendiente 
          .contenedor p {
        font-family: 'Arial';
    }
    • hijo directo > 
          .contenedor > p {
        font-size: 45px;
    }
    • hermano general ~ 
          p ~ figure {
        width: 100%;
    }
    • hermano adjacente + 
          img + figcaption {
        font-weight: bold;
    }

    • selector de atributos [atributo="valor"]

      • podemos seleccionar un elemencto especificando su atributo en caso de que tengamos varios y solo necesitemos uno en especifico 
                a[href="nosotros"]{

        }
      • el * selecciona al elemento que no imporat donde pero si contiene una palabra en especifico, sera seleccionado 
                a[href*="inicio"]{

        }
      • el ^ selecciona el elemento que en su atributo especificado comience con el texto definido 
                a[href^="http"]{

        }
      • el $ selecciona el elemento que en su atributo especificado termine con el texto definido 
                a[href$=".com"]{

        }
      • ~ no se en que se diferencia 
                a[class~=""]{

        }
      • el | se aplicara estilos si la palabra que buscamos tiene segido de ella un - 
                a[class|="menu"]{

        }
