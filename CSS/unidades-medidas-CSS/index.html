Custome Propertis (Variables CSS)

  • es una variable que almacena valores que se usaran en diferentes propiedades, una variable se declara comensando con --, y se utiliza dentro de var(--variable) 
    
    ::root {
        --bg-section: #fff;
    }
    section {
        background-color: var(--bg-section);
    }

Funciones CSS

  • var() -- sirve para utilizar nuestras custom propertis
  • url() -- sirve para agregar ruta a recursos
  • calc() -- sirve para hacer calculos arigmeticos (es importante separar con espacios los operandos del operador) width: calc(var(--width-box) + (var(--padding-size) * 2));
  • min() & max() width: min(300px, 20vw); //toma el valor mas chico de la "lista" height: max(200px, 25vh); //toma el valor mas grande de la "lista"
  • clamp() --esta funcion recibe 3 valores (Tama침o min, Tama침o ideal, Tama침o maximo) font-size: clamp(2rem, 1rem + 3vw, 3rem); // es muy buena obssion parael responsive design

estilos de texto

  • text-aling -- nos permite alinear el texto (left | center | right)
  • text-decoration -- podemos agregar siertas caracteristicas a los textos (lineas abajo, arriba, etc)
  • text-indent -- nos permite dar sangrita al texto
  • text-overflow -- si el texto se sale del contenedor podemos hacer que se apliquen unos puntos suspensivos (...)
  • white-space -- es el que permite que se agan los saltos de linea cuando el texto no cabe en la pantalla, su valor por default es "wrap"
  • text-transform -- nos permite transformar los textos de muchas formas
  • letter-spacing -- manipula el espacio entre letras
  • word-spacing -- manipula el espacio entre palabras
  • word-break -- nos permite romper palabras en caso de que se necesite
  • writing-mode -- manipula la amnera en que se presenta el texto
