estructura basica del documento HTML5

<!DOCTYPE html> <!--le indica al navegador la version de html que se esta usando-->
<html lang="en"><!--es la etiqueta raiz, todo nuestro codi html debe ir dentro de esta etiqueta-->
    <head>
    <!--es la etiqueta contenedora para todo lo que usa el navegador, no es visible para el usuario-->
        <meta charset="UTF-8">
        <!--habilita caracteres especiales (tildes, asentos,etc)-->
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <!--adacta el documento para representarlo en el navegador IE-->
        <meta name="viewport" content="width=device-width, initial-scale=1.0">
        <!--trata el documento para que el contenido sea "responsivo", en diferentes pantallas-->
        <title>Document</title>
        <!--titulo de nuestra pagina, entre 55 / 65 caracteres-->
        <meta name="description" content="esta pagina es una vista del curso de HTML de @jonmircha">
        <!--descripcion de nuestra pagina que aparecera en los navegadores, no mas de 156 caracteres-->
    </head>
    <body><
    !--todo lo visual para el usuario va dentro de la etiqueta body-->

    </body>
</html>

etiquetas mas usadas

la primera de ellas y sin duda la mas usada es <p>, con ella podemos crear parrafos en nustros sitios, y un parrafo es cualquier cosa que contenga texto.

otras equitetas muy usadas, son las etiquetas para dar formato a nuestro texto

  • b - texto en negritas

  • sup - texto sup indice
  • sub - texto sub indice

  • small - texto pequeño

Estructura semantica

etiquetas contenedoras semanticas segun su contenido
header - encabezado
nav - barra de navegacion / menu
main - contenido principal
section - contenido generico
article - contenido que por si solo se explica (seccion de auto-contenido)
aside - contenido secundario
footer - pie de pagina
