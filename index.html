estructura basica del documento HTML5

<!DOCTYPE html> <!--le indica al navegador la version de html que se esta usando-->
<html lang="en"><!--es la etiqueta raiz, todo nuestro codi html debe ir dentro de esta etiqueta-->
    <head><!--es la etiqueta contenedora para todo lo que usa el navegador, no es visible para el usuario-->
        <meta charset="UTF-8"><!--habilita caracteres especiales (tildes, asentos,etc)-->
        <meta http-equiv="X-UA-Compatible" content="IE=edge"><!--adacta el documento para representarlo en el navegador IE-->
        <meta name="viewport" content="width=device-width, initial-scale=1.0"><!--trata el documento para que el contenido sea "responsivo", en diferentes pantallas-->
        <title>Document</title><!--titulo de nuestra pagina, entre 55 / 65 caracteres-->
        <meta name="description" content="esta pagina es una vista del curso de HTML de @jonmircha"><!--descripcion de nuestra pagina que aparecera en los navegadores, no mas de 156 caracteres-->
    </head>
    <body><!--todo lo visual para el usuario va dentro de la etiqueta body-->

    </body>
</html>
